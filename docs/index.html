The sweep

A search engine that drives a real browser.

eDreams has no public API and blocks direct requests. So instead of faking one, the extension does what you'd do by hand — just tirelessly, and in order.

01

Generate the combinations

Every departure day in your window is paired with every nights value in your range. A 20-day window × 4 nights options is 80 searches.

02

Open each search in a real tab

One at a time, with a randomised 8–15 s delay between them — human-paced, so eDreams keeps serving real prices.

03

Read the Cheapest price off the page

Anchored on the page's own Cheapest tab, confirmed across a few readings so the number is the final one, not the optimistic first draft.

04

Rank, save, export

Results land in a sortable table, cheapest-first, saved to history, exportable as CSV.

What's inside

Built for accuracy, not just speed.

Dates

Flexible sweep

Departure window × nights range. Every valid round-trip, generated automatically.

Precision

Reads the real price

Anchored on the Cheapest tab and confirmed across readings — no optimistic placeholder, no guessed selector.

Timing

Self-correcting ETA

Time remaining is measured from real per-search timings, so it adjusts as the sweep goes.

Safety

Anti-bot aware

Sequential tabs, randomised delays, and an automatic pause if a CAPTCHA appears.

History

Every scan saved

Reopen any past sweep with its full results. Export any of them to CSV.

Details

Passengers & languages

Adults, children, infants. Interface in English or Portuguese, switchable anytime.

Get it

Two ways to install.

From source

  1. Download the repo from GitHub.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the folder.
  4. Click the icon to open the side panel.
